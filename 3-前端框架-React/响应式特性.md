#### **对比 Vue 与 React**：

| 特性     | Vue 3 [[vue响应式原理]]          | React                           |
| ------ | ------------------------ | ------------------------------- |
| 响应式核心  | `reactive`/`ref` + Proxy | `useState`/`useReducer` + 不可变更新 |
| 更新触发方式 | 自动（直接修改数据即可）             | 手动（需调用 setState）                |
| 依赖追踪   | 自动（模板/计算属性）              | 手动（依赖数组）                        |
| 底层实现   | Proxy（Vue 3）             | 闭包 + 调度算法（Fiber）                |
