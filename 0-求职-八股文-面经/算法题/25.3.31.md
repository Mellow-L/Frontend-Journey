#二分-easy
- JS中二分 / 操作可能产生浮点数，**改用位运算**
```js
var searchInsert = function(nums, target) {
    const n = nums.length
    let lb,ub,mid
    lb = 0
    ub = n-1
    while(lb<=ub){
        //mid = (lb+ub)/2
        mid = (lb+ub)>>1
        if(target <= nums[mid]){
            ub = mid - 1
        }else{
            lb = mid + 1
        }
    }
    return lb
};
```

#DP-easy
- 状态转移方程
	- `dp[i] = dp[i-1] + dp[i-2]`
	- 如果你到达 `n` 级台阶，有两种可能：
		- 从 `n-1` 级台阶爬 1 级（情况总数 `dp[n-1]`）。 
		- 从 `n-2` 级台阶爬 2 级（情况总数 `dp[n-2]`）。
```js
var climbStairs = function(n) {
    const dp = []
    dp[0] = 1
    dp[1] = 1
    for(let i = 2;i <= n;i ++){
        dp[i] = dp[i-1] + dp[i-2]
    }
    return dp[n]
};
```
