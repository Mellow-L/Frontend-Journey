| 分类   | 符号/用法               | 含义例子                           |
| ---- | ------------------- | ------------------------------ |
| 字符匹配 | `.`                 | 匹配任意单个字符（换行除外）                 |
| 字符类  | `[abc]`、`[^abc]`    | 匹配a或b或c / 除了a、b、c以外            |
| 范围   | `[a-z]`、`[0-9]`     | 匹配a-z或0-9之间的字符                 |
| 量词   | `*`、`+`、`?`、`{n,m}` | 0次以上 / 1次以上 / 0或1次 / n到m次      |
| 位置   | `^`、`$`             | 开头 / 结尾                        |
| 转义   | `\.`、`\d`、`\w` 等    | 匹配 . 字符本身；`\d` 数字；`\w` 字母数字下划线 |
| 分组捕获 | `()`                | 提取子匹配                          |
| 非捕获组 | `(?:...)`           | 仅用于分组，不捕获结果                    |
| 或关系  | `a                  | b`                             |
